Code Injection Attack

Injections are no fun and this one is even worse! Someone is injecting commands to send our rides crazy! Can you help identify our attacker and protect us from any injection attacks?

Click on each box to find the information:

A Code Injection attack consists of adding code that is executed by the server. They can do this by finding a gap or a loophole in the victim's computer. This can change how the program acts or gives control to the attacker.

This attack can cause the system to act differently, such as the motors speeding up

Code Injection attacks can be identified through network analysis. We need to find a packet going to a IP address and telling it to do something that it shouldnt do.
